## Setup

Install [stow](https://www.gnu.org/software/stow/)

```sh
sudo pacman -S stow
```

Clone this repository

```sh
git clone https://github.com/edddieee/dotfiles $HOME/.dotfiles && cd $HOME/.dotfiles
```

Set all symbolic links using stow

```sh
stow .
```
